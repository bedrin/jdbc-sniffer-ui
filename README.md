# sniffy-ui
Sniffy UI 

## Building

```
npm install
bower install
grunt
```

## Developing

```
grunt watch
```

## Testing with mock server

```
node index.js
```

Now open [http://localhost:3000/mock/mock.html](http://localhost:3000/mock/mock.html) in your browser and have fun!
